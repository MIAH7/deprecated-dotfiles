## Setup
* Linux distribution - Arch Linux
* Shell - ZSH
* Window Manager - SwayWM
* Bar - Waybar
* Application Launcher - wofi
* Notifications - Mako
* Terminal - Alacritty
* Text Editor - NeoVim
* Screenshot utility - Grim + Slurp + Swappy
